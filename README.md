# NeuralBlockForge

🚀 Build, train, and run AI models in your browser like playing with blocks, powered by WebGPU and TensorFlow.js! 🧠

`NeuralBlockForge` is a static website that allows you to build your own image classification AI models with an intuitive, visual programming interface, right in your browser. No deep coding knowledge required.

## ✨ Try it now!

You can try it out right away at the link below!

[**Experience NeuralBlockForge now**](https://kingyo1205.github.io/NeuralBlockForge/)

## 🌟 Features

*   **Visual Modeling:** Design your AI architecture simply by connecting blocks, powered by Blockly.
*   **High-Speed Training:** Leverage GPU acceleration for faster model training in browsers that support WebGPU.
*   **All in the Browser:** Everything from model definition, training, inference, and saving is done entirely in your browser. No installation needed.
*   **Open Source:** Published under the MIT License, allowing anyone to freely use, modify, and redistribute it.
